# budget-analyser

A React Native (Expo) mobile app for personal finance tracking — log transactions, set budgets, and view spending analytics.

## Features

- Dashboard overview of income vs. spending
- Transaction log with categories
- Budget targets per category
- Analytics screen with charts

## Setup

```bash
npm install
npx expo start
```

Scan the QR code with Expo Go on your phone, or press `a` / `i` to open an Android emulator or iOS simulator.

## Stack

- React Native + Expo
- React Navigation (bottom tabs + stack)
- AsyncStorage for local persistence
